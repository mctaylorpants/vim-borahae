## Installation
Use [vim-plug](https://github.com/junegunn/vim-plug)? Add the following lines to your `.vimrc` file (or `init.vim` if you use NeoVim):

```
Plug 'mctaylorpants/vim-borahae'
```

Then, enable the colour scheme:

```
colorscheme borahae
```
